# Social-Lorem-RN-Expo

Recreation of Social-Lorem w/ React-Native with Expo
This project uses the Social-Lorem-API (Graphql version)

# Features

- You can create and sign in a account
- Create a post
- Delete a post
- Comment on a post
- Like a post
- View the comments of a post
- Search for a user
- Infinite scroll
